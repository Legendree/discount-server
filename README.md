# Discount Application Programming Interface

Discount server that will handle users, posting, will have admin permissions, roles for the users,
ability to comment, like the posts that will automaticaly add those to favorites of the user.

###### Basic usage

How to commit to this repository correctly:

```
git add -A
git commit -m "Commit message"
git push
```

###### Be aware of

You can view the status of your local git repository on the machine with `git status`

If you need to view history of the changes in the repository simply use `git log`

Before doing anything new always be up-to-date with `git pull`
**and** use `npm update` to download all of the missing packages, if any were added.

If something doesn't work always check if **config.env** that is being passed around is up-to-date
